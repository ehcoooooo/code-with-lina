# Todo List App

A simple and elegant todo list application built with Next.js.

## Features

- Add new todos
- Mark todos as completed
- Delete todos
- Todos are persisted in localStorage
- Responsive design
- Clean and modern UI

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## How to Use

- Type your todo item in the input field and press Enter or click "Add"
- Click on a todo to mark it as completed/incomplete
- Click the "Delete" button to remove a todo
- Your todos will be saved automatically in your browser's localStorage

## Technologies Used

- Next.js
- React
- JavaScript
